# Klaystagram
Klaytn-Based NFT Photo Licensing Application Tutorial

## Installation
First, clone klaystagram repository.
```
$ git clone https://github.com/underbleu/klaystagram.git
```

Then install the package and run klaystagram app in browser.
```
$ npm install
$ npm run local
```
